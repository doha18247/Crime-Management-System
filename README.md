Collaborated on a team project to develop a Crime Management System using Java and JavaFX, designed to streamline law enforcement workflows. The system enables efficient tracking of cases, suspects, victims, witnesses, and evidence, with role-based access for admins, officers, and managers.

Highlights:

Case Management: Create, update, and monitor criminal cases with unique IDs.

Evidence Tracking: Manage evidence efficiently, linked to specific cases.

Criminal Profiling: Assess danger levels (High, Moderate, Low) based on criminal records.

User Roles: Admins oversee operations, officers handle case details, managers manage assignments.

Persistent Storage: Secure data handling via file I/O operations.

Built with object-oriented design, featuring modular classes for citizens, officers, criminals, and cases, ensuring scalability and maintainability. The JavaFX interface provides a seamless and intuitive user experience.
